# JoPoScrAgent — Job Postings Scraping Agent

**Что это**  
JoPoScrAgent — это **личная система для автоматического поиска вакансий**: она по расписанию собирает объявления с hh.ru, **отсекает дубликаты** (когда вакансию «подтолкнули» пустыми правками), **оценивает релевантность** вашему резюме, **принимает решения по порогам** (что смотреть, что в приоритете) и **уведомляет** вас в почте или мессенджере. Саму отправку отклика в документах оставляют за человеком — остальное вокруг отбора вакансий стараются автоматизировать.

**Для кого**  
Для **соискателя / разработчика (или специалиста в IT)**, который хочет тратить **не 2–3 обхода сайта в день вручную**, а **до ~30 минут** на обзор отобранного, с **локальным** хранением данных и **простым запуском** (в описании — через Docker).

**Какую задачу решает**  
Поиск работы на hh.ru **без спама из одних и тех же вакансий** и **без пропуска действительно новых** релевантных позиций, плюс **быстрая реакция** на сильные совпадения (в целевой картине — в разумных сроках после публикации/изменения) и **единая база** просмотренных вакансий и решений, чтобы **настраивать стратегию откликов** по фактам, а не по памяти.

**В двух фразах**  
Меньше рутины с hh.ru, меньше повторов и «шума» от фиктивных обновлений, больше **фокуса на вакансиях, которые реально вам подходят**, с прозрачной историей и уведомлениями — **без ухода ваших данных в облако** в базовом замысле из документации.
